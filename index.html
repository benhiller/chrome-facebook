acebook for Chrome by benhiller

Description

This extension lets you view your Facebook news feed, wall and notification all in a Chrome browser action. You can also post status updates by clicking on your profile picture. It looks like this.

Install

You can install the extension in Chrome by clicking here.It works correctly on Mac, seems to have issues on Linux and has not been tested on Windows (though most of the people using it are on Windows, so hopefully it works there).

You can also clone the project with Git by running: 

$ git clone git://github.com/benhiller/chrome-facebook

Get the source code on GitHub: benhiller/chrome-facebook
